## الشاشات المتاحة

### 🎬 شاشة البداية (Splash Screen)
- تظهر لمدة 3 ثوان مع شعار التطبيق
- تنتقل تلقائياً إلى شاشة تسجيل الدخول

### 🔐 شاشة تسجيل الدخول
**للتجربة:**
- أدخل أي بريد إلكتروني وكلمة مرور
- أو اضغط "تصفح كزائر" للدخول مباشرة

**أنواع الحسابات للتجربة:**
- `admin@test.com` → حساب مدير
- `company@test.com` → حساب شركة
- `pharmacy@test.com` → حساب صيدلية
- أي بريد آخر → حساب فردي

### 📝 شاشة التسجيل
- اختر نوع الحساب (فرد/شركة/صيدلية)
- املأ البيانات المطلوبة
- سيتم إنشاء الحساب تلقائياً

### 🏠 الشاشة الرئيسية
**المميزات:**
- شريط بحث في الأعلى
- زر البحث المتقدم (أيقونة الفلتر)
- تصنيفات سريعة قابلة للتمرير
- قائمة الأدوية الشائعة
- شريط تنقل سفلي (الرئيسية، التصنيفات، المفضلة، الحساب)

**التنقل:**
1. **الرئيسية**: الشاشة الرئيسية مع البحث والأدوية
2. **التصنيفات**: جميع المجموعات الدوائية + دليل الشركات
3. **المفضلة**: الأدوية المحفوظة
4. **الحساب**: الملف الشخصي والإعدادات

### 📋 شاشة التصنيفات
**للوصول:** اضغط على "التصنيفات" في الشريط السفلي

**المحتوى:**
- Antibiotics - المضادات الحيوية (12 فئة)
- Anticoccidials - مضادات الكوكسيديا (7 فئات)
- Anthelmintics - مضادات الديدان (13 فئة)
- Vitamins - الفيتامينات (6 فئات)
- Vaccines - اللقاحات (10 فئات)
- Miscellaneous - متنوعات (10 فئات)
- Feed & Water Additives
- Pet Animal Drugs

### 🏢 شاشة قائمة الشركات
**للوصول:** من شاشة التصنيفات → "عرض قائمة الشركات"

**المميزات:**
- بحث عن الشركات
- معلومات الشركة (الاسم، الدولة)
- زر "أضف شركتك"

### 💊 شاشة تفاصيل الدواء
**للوصول:** اضغط على أي دواء من القائمة

**المحتوى:**
- صورة الدواء مع التصنيف
- معلومات سريعة (الشركة، التعبئة)
- 4 تبويبات:
  - الاستخدام
  - الجرعة
  - الآثار الجانبية
  - التحذيرات
- زر المفضلة (قلب)
- زر المشاركة
- معلومات الشركة المصنعة
- أدوية مشابهة

### ⭐ شاشة المفضلة
**للوصول:** اضغط على "المفضلة" في الشريط السفلي

**المحتوى:**
- قائمة الأدوية المحفوظة
- إمكانية الحذف من المفضلة

### 👤 شاشة الملف الشخصي
**للوصول:** اضغط على "الحساب" في الشريط السفلي

**المحتوى:**
- معلومات المستخدم
- الملف الشخصي
- المفضلة
- سجل البحث
- الإشعارات
- الإعدادات
- عن التطبيق
- المساعدة
- اتصل بنا
- تسجيل الخروج

## نصائح للتطوير

### Hot Reload
اضغط `r` في Terminal لإعادة تحميل التطبيق بسرعة

### Hot Restart
اضغط `R` في Terminal لإعادة تشغيل التطبيق كاملاً

### تشغيل على أجهزة محددة
```bash
# عرض الأجهزة المتاحة
flutter devices

# تشغيل على جهاز محدد
flutter run -d <device-id>
```

## المشاكل الشائعة

### مشكلة: لا تظهر الأيقونات
**الحل:**
```bash
flutter clean
flutter pub get
flutter run
```

### مشكلة: خطأ في المكتبات
**الحل:**
```bash
flutter pub upgrade
```

### مشكلة: خطأ في Gradle (Android)
**الحل:**
```bash
cd android
./gradlew clean
cd ..
flutter run
```
