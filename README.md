# DrugVet - دواء 🏥

منصة إلكترونية متكاملة لعرض معلومات الأدوية البيطرية

## نظرة عامة

DrugVet هو تطبيق Flutter حديث يوفر منصة شاملة لعرض وإدارة معلومات الأدوية البيطرية. يهدف التطبيق إلى تسهيل الوصول إلى المعلومات الدوائية للأطباء البيطريين، الصيدليات، والشركات المصنعة.

## المميزات الرئيسية ✨

### 1. نظام المستخدمين
- **حسابات فردية**: للأطباء البيطريين والمستخدمين العاديين
- **حسابات الشركات**: للشركات المصنعة للأدوية
- **حسابات الصيدليات**: للصيدليات البيطرية
- **لوحة تحكم الإدارة**: لإدارة المحتوى والمستخدمين

### 2. البحث المتقدم 🔍
- البحث بالاسم التجاري
- البحث بالمادة الفعالة
- البحث بالشركة المصنعة
- البحث بالاستخدام
- تصفية حسب الفئات

### 3. تصنيف الأدوية 📋
- مضادات حيوية
- فيتامينات
- مطهرات
- مضادات الطفيليات
- هرمونات
- لقاحات
- مسكنات
- مضادات التهاب

### 4. معلومات شاملة عن كل دواء
- الاسم التجاري (عربي/إنجليزي)
- المادة الفعالة
- الشركة المصنعة
- الاستخدامات
- الجرعات
- الآثار الجانبية
- التحذيرات
- فترة السحب

### 5. ميزات إضافية
- حفظ الأدوية المفضلة ⭐
- مشاركة معلومات الأدوية
- واجهة مستخدم حديثة وسهلة الاستخدام
- دعم اللغتين العربية والإنجليزية

## الشاشات المتوفرة 📱

### 1. شاشة البداية (Splash Screen)
- عرض شعار التطبيق مع تأثيرات حركية
- انتقال تلقائي بعد 3 ثوان

### 2. شاشة تسجيل الدخول
- تسجيل الدخول بالبريد الإلكتروني وكلمة المرور
- خيار التصفح كزائر
- رابط لإنشاء حساب جديد
- نسيت كلمة المرور

### 3. شاشة التسجيل
- اختيار نوع الحساب (فرد/شركة/صيدلية)
- نموذج تسجيل شامل
- التحقق من صحة البيانات
- حقول ديناميكية حسب نوع الحساب

### 4. الشاشة الرئيسية
- شريط بحث متقدم
- تصنيفات سريعة بألوان مختلفة
- قائمة الأدوية الشائعة
- شريط التنقل السفلي (4 أقسام)
- نافذة البحث المتقدم المنبثقة

### 5. شاشة التصنيفات
- جميع المجموعات الدوائية
- الضغط المباشر على التصنيف يعرض الأدوية
- عرض الأدوية في شبكة 2×N
- أكثر من 11 تصنيف رئيسي

### 6. شاشة قائمة الشركات
- دليل شامل للشركات
- بحث عن الشركات
- معلومات الاتصال
- خيار إضافة شركتك

### 7. شاشة تفاصيل الشركة
- معلومات الشركة الكاملة
- منتجات الشركة
- معلومات الاتصال
- أزرار الاتصال المباشر

### 8. شاشة تفاصيل الدواء
- معلومات شاملة عن الدواء
- 4 تبويبات منظمة
- إمكانية الحفظ في المفضلة
- مشاركة المعلومات
- أدوية مشابهة
- معلومات الشركة المصنعة

### 9. شاشة المفضلة
- قائمة الأدوية المفضلة
- إمكانية الحذف من المفضلة
- الوصول السريع للأدوية

### 10. شاشة الملف الشخصي
- معلومات المستخدم
- قائمة الإعدادات
- سجل البحث
- الإشعارات
- عن التطبيق
- تسجيل الخروج
